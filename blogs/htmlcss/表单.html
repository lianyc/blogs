    表单相关

    10/23/2023 表单

    # 1.使用form替代div包裹表单

    ​ form自带回车提交，可响应type="submit"的事件

    <form>
    <input type="test" />
    <input type="submit" value="提交" />
</form>

    # 2.select多选

    <select multiple>
    <option>1</option>
    <option>2</option>
    <option>3</option>
</select>

    # 3.radio的值与显示文本点击关联

    • 指定label标签的for属性
    <input type="radio" name="sex" id="male"/>
<label for="male">男</label>
    • 使用label标签包裹
    <label>
	<input type="radio" name="sex"/>
	<span>男</span>
</label>

    # 4.重置表单

    <form>
    <button type="reset">重置</button>
</form>

    # 5.Textarea禁止拖动改变大小

    textarea {
    resize: none;
}

    # 6.浮动布局以及清除浮动

    .left {
    float: left;
}
.right {
    float: right;
}
.clearfix::after {
    content: '';
    display: block;
    clear:both;
}

    # 7.两个行级元素垂直对齐

    .inline-div {
    vertical-align: 3px;
}

    # 8.选中后面的兄弟元素

    div ~ span 选中后面所有的span
div + span 选中后面第一个span

    # 9.placeholder样式修改

    input::placeholder {
    color: red;
    font-size: 16px;
}

input:focus::placeholder {
    color: red;
    font-size: 16px;
}
